
# 💈 Benyita Barber

Sistema de reservas online para la barbería **Benyita Barber**.

## 🚀 Funciones
- Agendar horas online
- Bloqueo de horarios ocupados
- Horarios automáticos:
  - Semana: 16:00 - 19:00
  - Fin de semana: 11:00 - 19:00
- Conectado a base de datos (Firebase)

## 📱 Uso
Entra a la página y:
1. Ingresa tu nombre
2. Selecciona fecha
3. Elige hora disponible
4. Agenda tu reserva
